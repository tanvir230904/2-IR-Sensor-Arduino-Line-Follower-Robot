# 2-IR-Sensor-Arduino-Line-Follower-Robot
A simple autonomous line-following robot using two IR sensors and an Arduino motor shield. The robot detects black and white surfaces and adjusts motor movement to follow a black line.
Features
Two IR sensor line detection
Automatic left and right turning
Forward movement on track
Stop condition when both sensors detect black
Adjustable speed parameters
Serial Monitor debugging output
Components Used
Arduino Uno
Adafruit Motor Shield
2 × IR Sensors
4 × DC Motors
Robot chassis
Wheels
Battery pack
Jumper wires
